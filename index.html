WCAG Lookup Tool

An interactive semantic search overlay designed for digital accessibility specialists to reference compliance criteria, failures, and disability impacts directly on the page being audited.

Download Bookmark Shortcut File

Installation Instructions

  1. Activate the "Download Bookmark Shortcut File" button above to save the installation file directly to your system's downloads folder.
  2. Open Chrome's full-screen Bookmark Manager by pressing Ctrl + Shift + O (Windows) or Cmd + Option + B (Mac).
  3. Navigate to the Organize Menu button (the three vertical dots located in the upper-right corner of the manager view) and activate it.
  4. Select Import bookmarks from the dropdown list options.
  5. Choose wcag-bookmark-import.html from your local downloads folder and select open.

How to Run the Tool

Navigate to any web domain you want to evaluate. Open your bookmarks folder structure, choose the newly imported WCAG Lookup Tool link, and press Enter. The visual search panel will slide directly onto the page window layout.

⌨️ Keyboard Efficiency Tip: You can launch this completely hands-free. Press Alt + F then B to enter your bookmarks menu, then simply type WCAG to jump focus directly to the tool and activate it. Once launched, toggle panel visibility instantly at any point using Alt + Shift + A.
